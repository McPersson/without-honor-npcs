# Without Honor: NPCs

Friendly shopkeepers. Quest givers with branching stories. Town guards, bosses,
wandering travelers with their own daily routines. Without Honor: NPCs lets you build
all of them.

Everything happens in-game. Grab the Memorion Feather, right-click an NPC, and a full
editor opens up: dialogue trees, shops, reputation, combat, poses, schedules - all
through a clean (almost :D) GUI.

A Minecraft **Forge 1.20.1** mod. Currently in **beta (0.9.2)**.

**Download:** [Modrinth](https://modrinth.com/mod/without-honor-npcs) · [CurseForge](https://www.curseforge.com/minecraft/mc-mods/without-honor-npcs) - **Report a bug:** [Issues](../../issues)

## Features at a glance

- **Branching dialogues** with conditions, actions, a glossary, images, a tall 9:16
  portrait, a second on-screen character, per-node music and per-line sound cues.
- **A living world**: daily schedules, a pose library with animation freeze, emotes
  (via Emotecraft), event reactions and random ambient lines.
- **RPG systems**: reputation & factions, per-player flags that remember your quest
  state, and deep villager-style trading with limits, conditions and restocks.
- **Combat & creature nature**: combat styles, seven damage resistances, an optional
  boss health bar, disguise an NPC as any mob, and a death & respawn system.
- **Tools**: an in-game editor (the Memorion Feather), a clone book, an NPC spawner
  block, hidden triggers, and file export/import for profiles and dialogues.

For the full description and screenshots, see the
[Modrinth](https://modrinth.com/mod/without-honor-npcs) or
[CurseForge](https://www.curseforge.com/minecraft/mc-mods/without-honor-npcs) page.

## Requirements

- Minecraft **1.20.1**, **Forge**.
- Install on **both client and server** (custom NPC rendering and the editor need the
  mod client-side).
- Optional integrations (all soft dependencies): Emotecraft, Curios, Etched, Epic Fight, Carry On.
